captureVisibleTab 利用 chrome api 进行截图

当我们打开 chrome（版本 >= 59），chrome 浏览器本身有截图功能，支持三种截图：
打开 chrome 的开发工具 (F12 或 ctrl + shift + i), 然后 ctrl + shift + p

  • capture full size screenshot 截取长图
  • capture node screenshot 节点截图
  • capture screenshot 截取可是区域
    刚看到的时候还挺惊喜的，毕竟我总是没有见过世面的样子，立刻去 B 站把三种截图的试了一遍，感觉还不错。

但是本篇我们要介绍的是 chrome 扩展实现截图功能，完美一点的就类似 花瓣 的采集工具， 印象笔记 的剪藏工具也不错。

主要用到的 就是 chrome.tabs.captureVisibleTab、制作简单的扩展的一些基本知识 以及 一些 chrome 扩展的 api, 比如 （页面通信、后台常驻、跨域请求等）。本篇只会实现一个简单的 demo 。
